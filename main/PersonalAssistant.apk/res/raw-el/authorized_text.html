Ο Προσωπικός Βοηθός είναι ένα σημείο αναφοράς για όλες τις πιο χρησιμοποιούμενες λειτουργίες και πληροφορίες. Για να μπορείτε να δουλεύετε κανονικά, ο Προσωπικός Βοηθός πρέπει να συνδεθεί στο διαδίκτυο, καθώς και να χρησιμοποιήσει τις λειτουργίες και τα δικαιώματα άλλων εφαρμογών που είναι εγκατεστημένες στη συσκευή σας. Για παράδειγμα, ο Προσωπικός Βοηθός πρέπει να διαβάσει τα μηνύματα κειμένου και τα ημερολόγια σας για να μπορέσει να σας δώσει πληροφορίες σχετικά με τα ταξίδια, τις πτήσεις σας και τις κρατήσεις ξενοδοχείων σας. Ωστόσο, τα μηνύματά σας και τα συμβάντα ημερολογίου δεν θα φορτωθούν στους διακομιστές μας και θα υποβληθούν σε τοπική επεξεργασία στη συσκευή σας. Ενδέχεται επίσης να χρειαστεί να συλλέξουμε και να χρησιμοποιήσουμε τις ακόλουθες πληροφορίες από εσάς (ανατρέξτε στην "Πολιτική Απορρήτου Προσωπικού Βοηθού" για λεπτομέριες).

Πληροφορίες συσκευής: Οι πληροφορίες σχετικά με τη συσκευή σας, συμπεριλαμβανομένου του αριθμού IMEI, της διεύθυνσης MAC, του αναγνωριστικού Android, της έκδοσης συστήματος, της λίστας εφαρμογών, των πληροφοριών εφαρμογής κ.λπ..
Στοιχεία λογαριασμού: Οι αριθμοί τηλεφώνου που χρησιμοποιούνται για τις υπηρεσίες παράδοσης, τους αριθμούς παραγγελίας άμεσης παράδοσης, τις διευθύνσεις, τα ονόματα των υπηρεσιών συνδρομής, τις πληροφορίες εξουσιοδότησης, τα στοιχεία ταξιδιού κ.λπ..
Πληροφορίες τοποθεσίας: Μπορούμε να συλλέξουμε τα στοιχεία της τοποθεσίας σας (π.χ. συντεταγμένες) όταν χρησιμοποιείτε υπηρεσίες βάσει τοποθεσίας.
Πληροφορίες χρήσης: Οι πληροφορίες σχετικά με τον τρόπο που χρησιμοποιείτε τις εφαρμογές (π.χ. τα στοιχεία που αγγίζετε).

Ενδέχεται να μην είμαστε σε θέση να σας παρέχουμε υπηρεσίες σε περίπτωση που επιλέξετε να μην χορηγήσετε σχετικές άδειες ή να μην μας επιτρέψετε να συλλέξουμε και να χρησιμοποιήσουμε τις απαιτούμενες πληροφορίες. Ανοίξτε τον Προσωπικό Βοηθό, μετακινηθείτε προς τα κάτω, πατήστε \"Προσαρμογή\", και να καταργήσετε μεμονωμένες υπηρεσίες για να περιορίσετε τη συλλογή και τη χρήση των πληροφοριών που περιγράφονται παραπάνω.
